## Taipei Zoo App
> 串接台北動物園 API，顯示館區、動物、植物資訊

UI 設計：Figma<br>
App 架構：MVVM<br>
DI：Hilt<br>
Network：Retrofit、Moshi<br>
Database：Room<br>
Image: Coil
